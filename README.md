# Cinema Ticket Pricing Calculator

A lightweight Python script that calculates the final price of a movie ticket by evaluating multiple customer and showtime variables. This project demonstrates foundational programming concepts like conditional logic, Boolean operators, and dynamic pricing models.

##  Overview

The script processes a predefined set of user attributes (such as age and membership status) and show details (seat type, time, and day) to determine whether a user is eligible to book a ticket. If the criteria are met, it calculates the final ticket price by applying the appropriate base fares, surcharges, discounts, and service fees.

##  Features

* **Age-Based Access Control:** Validates if a user is old enough to book a ticket (18+) and handles specific age restrictions for evening shows (21+).
* **Dynamic Surcharges:** Automatically applies extra charges for weekend screenings or evening showtimes.
* **Membership Discounts:** Applies automated discounts for eligible club members.
* **Tiered Seating Fees:** Calculates service charges based on the selected seating tier (`Premium`, `Gold`, or `Standard`).
* **Robust Logic Flow:** Utilizes complex Boolean logic to ensure all business rules are satisfied before issuing a ticket.
